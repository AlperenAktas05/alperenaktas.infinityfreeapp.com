CSS NEDİR?

CSS, HTML dilinde yazılmış elementleri özelleştirmemizi sağlayan bir dildir.

CSS NASIL KULLANILIR?

CSS kodları, HTML sayfasında <style> etiketi içine yazılır. Bunun için 3 yol vardır:

  • Etiketin İçi: En öncelikli ve en ilkel CSS kodu yazış biçimidir. Yazılan kodlar bir süre sonra karmaşık hale gelir. Anlaşılamazlar.

See the Pen Untitled by Alperen Aktaş (@Alperen-Akta) on CodePen.

  • Dosyanın İçi: Orta öncelikli CSS kodu yazış biçimidir. Bütün CSS kodları <head> etiketi içerisine açılan <style> etiketine içerisine yazılır.

See the Pen CSS2 by Alperen Aktaş (@Alperen-Akta) on CodePen.

  • CSS Sayfasının İçi: En gelişmiş CSS kodu yazış biçimidir. Yazılan kodlar düzenli durur ve karışmaz. En son önceliğe sahiptir. Bunun için proje klasörü içine bir .css dosyası eklenir. Bu CSS dosyasını HTML dosyasıyla eşleştirmek için <link rel="stylesheet" href="dosya.css"> yazılır.

See the Pen CSS3 by Alperen Aktaş (@Alperen-Akta) on CodePen.

Son 2 stilde bütün CSS kodları aynı yere yazılır. Bu kodların hangi elemente uygulanması gerektiğini seçiciler ile ayırt ederiz. Yukarıda da görüldüğü üzere özellik atanması istenen etiketler yazılmış ve özellikler o etiketin parantezleri arasına yazılmıştır.

Seçiciler farklı şekillerde ifade edilebilir. Önceki konulardan ID ve Classlar burada önem kazanır.

CSS KUTU ÖZELLİKLERİ

Geldik zurnanın zırt dediği yere. Benim için CSS’in en önemli konusu çünkü artık her elementi hareket ettirip istediğimiz tasarımı gerçekleştirebiliriz. HTML’in tasarımsızlığından kurtuluyoruz.

HTML’de her şey birer kutudur. Daha öncesinde blok ve inline elementler konusunda bahsetmiştik. Biz bu kutuların boyutunu ve pozisyonlarını değiştirebiliriz.

  • İçerik: HTML kodlarımızın bulunduğu kısımdır.

  • Border: Kutunun sınırıdır.

See the Pen CSS15 by Alperen Aktaş (@Alperen-Akta) on CodePen.

  • Margin: Kutunun diğer kutular arası uzaklığını belirler.

See the Pen CSS16 by Alperen Aktaş (@Alperen-Akta) on CodePen.

Kutuların boyutunu ayarlarmak için ise width ve height özellikleri kullanılır.

See the Pen CSS17 by Alperen Aktaş (@Alperen-Akta) on CodePen.
